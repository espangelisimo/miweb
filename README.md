# Portafolio personal — Ángel Velarde

Portafolio profesional de Ángel Velarde, diseñador y desarrollador web especializado en negocios locales y sector salud en Sevilla (La Rinconada).

## Stack

HTML5 semántico · CSS3 con custom properties · JavaScript vanilla · Sin frameworks ni pasos de build · Compatible con GitHub Pages

## Estructura

```
/
├── index.html              # One-page principal
├── proyectos/
│   ├── fisiosalud.html     # Caso de estudio FisioSalud San José
│   └── clinilab.html       # Caso de estudio Clinilab Pastor
├── css/styles.css          # Único stylesheet organizado por bloques
├── js/main.js              # JS mínimo con progressive enhancement
├── img/                    # Imágenes en WebP
├── assets/                 # Favicon, og-image y otros assets
├── sitemap.xml
├── robots.txt
└── 404.html
```

## Despliegue

GitHub Pages (rama `main` o carpeta `/docs` según configuración).
